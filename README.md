# 🏧 ATM Management System

A simple ATM Management System built using **Python** and **Object-Oriented Programming (OOP)** concepts. This project simulates basic banking operations through a command-line interface.

---

## ✨ Features

✅ Create ATM PIN  
✅ Change PIN  
✅ Deposit Money  
✅ Withdraw Money  
✅ Check Account Balance  
✅ Secure Data using Encapsulation  
✅ Menu-Driven Interface  

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 🏗️ Object-Oriented Programming (OOP)
- 💻 Command Line Interface (CLI)

---

## 📚 OOP Concepts Implemented

- 🔹 Classes & Objects
- 🔹 Encapsulation
- 🔹 Private Variables
- 🔹 Methods
- 🔹 Static Methods
